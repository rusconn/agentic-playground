# アプリケーション仕様

## 概要

コマンドを通じてタスクリストを管理するCLI。

## コマンド一覧

### `add <title>`

- タスクを追加する
- タスクIDを発行する
- タスクの初期状態は未完了

### `ls`

- タスク一覧を表示する
- 表示内容にはid,title,doneが含まれる

### `done <id>`

- 指定したタスクを完了済みへ変更する

## タスクストレージ

- プロジェクトルートの`tasks.json`

### データ構造

```json
[
  {
    "id": number,
    "title": string,
    "done": boolean
  }
]
```
